# Aplicação de Visão Computacional

Esta é uma aplicação desenvolvida para explorar e demonstrar técnicas de visão computacional. O objetivo principal é permitir a análise e interpretação de imagens e vídeos, utilizando algoritmos modernos para tarefas como detecção de objetos, reconhecimento facial e segmentação de imagem.

## Funcionalidades

- **Detecção de Objetos:** A aplicação pode identificar e localizar objetos em imagens utilizando modelos treinados.
- **Reconhecimento Facial:** Integra tecnologia de reconhecimento facial para identificar pessoas a partir de imagens ou vídeos.
- **Segmentação de Imagem:** Permite isolar e identificar diferentes regiões em uma imagem, facilitando a análise de conteúdos específicos.

## Tecnologias Usadas

- Python
- OpenCV
- TensorFlow
- Keras
